Pre-flight checklist
contentBlocks coverage, h2/h3 diff, font uniqueness, bg-white, brand tab sync
GATE
Per-Article Font Accuracy
NEVER copy-paste font data between articles. Each article's fonts[] must be independently extracted from its own source HTML.
Per-Article Heading Styles
h2 and h3 MUST have different extracted values. If they appear identical, re-extract — it's a bug.
contentBlocks = Single Source of Truth
Every interactive element needs BOTH a metadata entry AND a working React component with data.
AI2HTML: Image Already Has Text
Text appears as PNG pixels AND HTML overlays. Default to Scenario A: plain <img> only.
bg-white Requirement
Article page must use bg-white, NOT bg-zinc-50. Known enforcement point.
Dynamic Brand Tab Aggregation
Tabs aggregate on every render from the ARTICLES array. No manual updates needed when articles are added.
