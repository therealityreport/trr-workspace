Critical Enforcement Rules
Per-Article Font Accuracy
NEVER copy-paste font data between articles. Every article must have independently extracted font families, weights, and sizes.
extract-css-tokens, generate-article-page
Per-Article Color Extraction
Do NOT assume all articles use the same palette. Extract actual computed colors from each page independently.
extract-css-tokens
Heading Style Differences
h2 and h3 MUST have different extracted values. If they appear identical, re-extract — common failure mode.
extract-css-tokens, generate-article-page
Consistent Row Count (Tables)
All dropdown options must be padded to maxRows with em-dash. Visual inspection of rendered rows required.
extract-birdkit-tables
contentBlocks = Single Source of Truth
Every interactive element needs BOTH a metadata entry AND a working React component with data. Metadata alone = empty page.
generate-article-page
AI2HTML "Image Already Has Text"
Text appears both as PNG pixels AND HTML overlays. Default to Scenario A: plain <img> only. Never duplicate text rendering.
extract-ai2html-artboards
Dynamic Tab Aggregation
Brand tabs don't cache — they aggregate dynamically from ARTICLES array on every render. No manual updates needed when articles are added.
sync-brand-page, generate-brand-section
bg-white Requirement
Article page MUST use bg-white, NOT bg-zinc-50. Known pitfall enforced at wiring step.
wire-config-and-routing
Medal Circle Colors
Gold #C9B037, Silver #A8A8A8, Bronze #AD8A56. Hex value verification required for any table with medal indicators.
extract-birdkit-tables
Type-Check Gate
npx tsc --noEmit must exit with 0 errors. Prevents deployment with type errors. Auto-fix attempted before failing.
wire-config-and-routing (Wave 5)
